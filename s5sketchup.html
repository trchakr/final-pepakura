SET UP SKETCHUP

SketchUp offers both a paid desktop version and a free browser-based version. If you have the paid version, great—dive right in! If not, the free browser version is a solid option to get started.

Here’s how to set it up:

  1. Start Modeling
    • Head to the free browser version by following this link: SketchUp Free
    • Once you are there, press the Start Modeling button.
  2. Create Your Account
    • You’ll need to create an account to save your progress and access features.

Once you're set up, you're ready to go to the next step!
